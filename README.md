# Shieldcore
**Shieldcore is a powerful anti-cheat framework for Roblox using Typescript**

**Shieldcore is:**
- **⚡Easy to use**
- **🚀Powerful**
- **⚙️Extensible**

# Installation
1. npm install shieldcore
2. Go into node_modules/shielcore
3. Copy the src/server/decorators directory into your src/server
